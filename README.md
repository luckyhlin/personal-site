## Personal Site

The site is built upon Next.js and based upon Notion as the CMS. See my `about` page for details.

Here are the supported blocks so far:

| Block Type         | Availability |
|--------------------|:-----------:|
| Paragraph          |     ✔️      |
| Heading            |     ✔️      |
| Bulleted Item List |     ✔️      |
| Numbered Item List |     ✔️      |
| Toggle             |     ✔️      |
| Callout            |     ✔️      |
| Inline Equation    |     ✔️      |
| Image              |     ✔️      |
| Video              |      ❌      |
| Divider            |     ✔️      |
| Table              |      ❌      |
| Page Reference     |      ❌      |
| Block Equation     |     ✔️      |
| Quote              |     ✔️      |
